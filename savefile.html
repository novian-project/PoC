Uji Coba File Picker (Ekstensi Tersembunyi)

Metode 1: Mode "All Files" Tanpa Objek Types

Dialog hanya menampilkan pilihan "All Files (*.*)". Ekstensi file (.xlsx) disisipkan langsung pada nama default.

Metode 2: Menggunakan MIME Type Umum (Octet-Stream)

Menghapus opsi "All Files" bawaan browser dan menggantinya dengan deskripsi kustom tanpa memunculkan daftar ekstensi aslinya (.csv).
